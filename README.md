# Enoca Assignment
### Technologies
    -Spring Boot
    -Spring Data Jpa
    -Spring Validation
    -Mysql
    -Docker
### Run Project
If you want to run this project, you must have installed Docker and Docker-compose on your system.
#### Step 1
 ``` 
 docker build -t enoca-assignment:1.0 .
 ```
#### Step 2
 ```
 docker-compose up
 ```
Project is run in 8085 port.
